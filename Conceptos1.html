Estructura HTML

Para aprender sobre la estructura de HTML5 de sus etiquetas y más, vamos a conocer un poco sobre ¿Qué es HTML?,¿Qué es HTML5? y ¿Cuáles son sus etiquetas o tags?

¿Qué es HTML?


Pinos, J. (2020). ¿Qué es HTML? [Figura]. Recuperado de https://devcode.la/blog/que-es-html/

"HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc." (desarrolloweb, 2001, p.1)

¿Qué es HTML5?

Desarrolladores (s.f). Estructura de una página html5 [Figura]. Recuperado de https://desarrolladoresweb.org/html5/estructura-de-una-pagina-html5/

Html5 es la versión actualizada de html5, es una versión bastante completa porque se compone de diversas etiquetas y herramientas que permiten un mejor desarrollo de una página web. "No se limita solo a crear nuevas etiquetas o atributos, sino que incorpora muchas características nuevas y proporciona una plataforma de desarrollo de complejas aplicaciones web mediante los APIs" (arkaitzgarro, s.f, p.1). Es una versión que aun se encuentra en proceso de desarrollo y que va a a reemplazar las versiones anteriores de HTML.

¿Cuáles son sus etiquetas?

Los tags o etiquetas son elementos que le dicen al navegador como debe formatear el texto para poder visualizarlo, hay que tener en cuenta que Html5 cuenta con una gran variedada de etiquetas las cuales permiten un mejor manejo e interacción con las páginas web. En este apartado vamos a estudiar las etiquetas más básicas de una página, las cuales son:

  • Section
  • Aside
  • Footer
  • Article
